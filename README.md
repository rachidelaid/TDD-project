# TDD project

In this project we implemented a class with some methods, but we did it by doing test-driven development (TDD). the idea is to write tests first and then the code.

## Built With

- Ruby
- rspec

### Setup <br>

> To clone Run `git clone https://github.com/rachidelaid/TDD-project.git`

<br>

> To goto the program `cd TDD-project`

<br>

> To run the test `rspec spec/solver_spec.rb -f d`
